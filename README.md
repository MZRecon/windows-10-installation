# windows-10-installation
“Step-by-step installation of Windows 10 on a Lenovo laptop using a bootable USB, including BIOS configuration and troubleshooting.”
Windows 10 Installation (From Windows 8)

📌 Overview

This project documents the process of upgrading a Lenovo laptop from Windows 8 to Windows 10 using a bootable USB.

🛠️ What I Did

- Created a bootable Windows 10 USB
- Accessed BIOS using Novo button
- Changed boot priority to USB
- Installed Windows 10 using custom installation
- Configured system after installation

🚧 Challenges Faced

- Boot menu not opening with F12
- Had to use Novo button instead
- BIOS settings configuration

✅ Outcome

Successfully installed Windows 10 and set up the system.

🧠 Skills Learned

- BIOS navigation
- Bootable USB setup
- Operating system installation
- Troubleshooting boot issues
